# Dissertation

## Overview

This repository accompanies my dissertation, focusing on evaluating various gradient computation methods within MuJoCo’s physics engine. The project is organized into three main directories:
	1.	GradientExperiments: Contains environments such as one-bounce, two-bounce, finger, and pusher setups. These are used to experiment with different gradient methods for MuJoCo’s solver, including automatic differentiation, finite differences, and implicit differentiation.
	2.	diff_sim_grads: A third-party repository by researchers analyzing gradient performance against analytical ground truths across various physics simulators (excluding MuJoCo). This serves as a foundational reference for my experiments.  ￼
	3.	MJX: A collection of code from my supervisor’s research group, facilitating policy optimization and reinforcement learning tasks within MuJoCo.
